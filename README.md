# 🚀 App Exemplo Antigravity - Vibe Coding para Serviço Público

Este repositório é um guia prático e template para alunos do curso **Vibe Coding para Serviço Público**. Aqui você encontrará a configuração ideal para trabalhar com o **Antigravity**, o agente de codificação de IA mais poderoso da atualidade.

[✨ Participe da Comunidade GPTGOV](https://gptgov.com.br/comunidade)

</div>

---

## 🎯 Sobre este Projeto

Este projeto foi estruturado com uma stack moderna (**React + TypeScript + Vite**) pré-configurada para que o Antigravity entenda perfeitamente o contexto da sua aplicação. É o ponto de partida ideal para transformar ideias em ferramentas reais de impacto no setor público.

### O que você encontrará aqui:
- **Configuração Vite otimizada**: Para builds rápidos e ambiente de desenvolvimento fluido.
- **Estrutura TypeScript**: Tipagem robusta para evitar erros comuns.
- **Integração com Antigravity**: Instruções e contextos que facilitam a criação de funcionalidades complexas por IA.

---

## 🛠️ Gerando seu PRD (Product Requirement Document)

Para aproveitar ao máximo o Antigravity, você precisa de um PRD bem estruturado. Os alunos do curso possuem crédito gratuito para testar no **PRD GPTGOV**.

👉 **Acesse:** [prd.gptgov.com.br](https://prd.gptgov.com.br)

### 💡 Sugestão de Prompt para o Gerador de PRD

Ao gerar o seu PRD para ser usado com este template, utilize o seguinte prompt no app:

> "Gere um PRD detalhado para uma ferramenta de [DESCREVA SUA IDEIA, EX: GESTÃO DE LICITAÇÕES]. A aplicação deve ser desenvolvida em **React (Vite + TypeScript)**. 
> 
> **Requisitos Específicos:**
> 1. Foque em usabilidade para o servidor público (acessibilidade, clareza e design premium).
> 2. Detalhe os componentes necessários (telas, modais, dashboards).
> 3. Defina a estrutura de dados (campos, tipos e validações).
> 4. O PRD deve ser otimizado para leitura por agentes de código autônomos (Antigravity), com instruções claras e modulares.
> 5. Inclua um guia de estilo visual moderno e profissional."

## � Requisitos do Sistema

Antes de começar, certifique-se de ter as ferramentas abaixo instaladas em sua máquina:

- **Node.js (v18 ou superior)**: É o ambiente de execução necessário para rodar o React e o Vite. [Baixe aqui](https://nodejs.org/).
- **Gerenciador de Pacotes**: O Node já vem com o `npm`, mas você também pode usar `yarn` ou `pnpm` se preferir.
- **Git**: Para clonar o repositório e gerenciar suas versões.

---

## �🚀 Como Começar Localmente

1. **Pré-requisitos:** Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. **Instalar Dependências:**
   ```bash
   npm install
   ```
3. **Rodar em Desenvolvimento:**
   ```bash
   npm run dev
   ```
4. **Abrir no Antigravity:** Aponte o agente para esta pasta e comece a construir as funcionalidades descritas no seu PRD gerado.
